##  A + B

### 题目

输入两个整数，求这两个整数的和是多少。

### 输入格式

输入两个整数A,B，用空格隔开，$0≤A,B≤10^8$

### 输出格式

输出一个整数，表示这两个数的和

### 数据范围



### 输入样例

```
3 4
```

### 输出样例

```
7
```

### AC代码

```c++
#include <iostream>
using namespace std;
int a;int b;

int main(){
    cin >> a >> b;
    cout << a+b;
    return 0;
}
```
